centos7安装虚拟环境 virtualenvwrapper

发表于2022-09-16|更新于2022-09-16|Linux
|阅读量:

个人安装环境

1.1 CentOS Linux release 7.6 (华为云)

安装步骤

查找python3命令路径

1、查找python3 路径

[root@localhost ~]# find / -name python3

我的路径为：/usr/bin/python3

2、查找virtualenvwrapper.sh脚本的路径

[root@localhost ~]# whereis virtualenvwrapper.sh
virtualenvwrapper: /usr/local/python3/bin/virtualenvwrapper.sh

3、 安装virtualenvwrapper

yum install python-setuptools python-devel
pip3 install virtualenvwrapper

编辑bashrc

[root@localhost ~]# vim .bashrc
添加如下内容：

4、应用如上修改

source ~/.bashrc

5、虚拟环境使用

1.1 新建python2环境

[root@localhost ~]# mkvirtualenv -p /usr/bin/python p2test

1.2 新建python3环境

[root@localhost ~]# mkvirtualenv p3test

1.3 查看所有的虚拟环境

[root@localhost ~]# workon

1.4 进入指定的虚拟环境

[root@localhost ~]# workon py3env
(py3env) [root@localhost ~]# python
Python 2.7.5 (default, Jun 28 2022, 15:30:04)
[GCC 4.8.5 20150623 (Red Hat 4.8.5-44)] on linux2
Type “help”, “copyright”, “credits” or “license” for more information.

(py3env) [root@localhost ~]# python3
Python 3.8.5 (default, Sep 12 2022, 10:14:31)
[GCC 4.8.5 20150623 (Red Hat 4.8.5-44)] on linux
Type “help”, “copyright”, “credits” or “license” for more information.

1.5 退出虚拟环境

(py3env) [root@localhost ~]# deactivate

1.6 删除虚拟环境

[root@localhost ~]# rmvirtualenv py3test
