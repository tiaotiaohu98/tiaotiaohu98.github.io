题目：给你一个整数数组 nums 。如果任一值在数组中出现 至少两次 ，返回 true ；如果数组中每个元素互不相同，返回 false 。

解法一：穷举

  • 第一步：num[j]从num[0]开始遍历，遍历整个数组看是否有和num[0]相等的元素，若有，返回true，否则进行第二步
  • 第二步：遍历整个数组和数组第二个元素num[1]进行比较，若有，返回true，否则继续前两步
  • 第三步：直到num[i]=nums.length，若一直没有则返回false
public boolean containsDuplicate(int[] nums) {
        for(int i=0;i<nums.length;i++){
            for(int j=i+1;j<nums.length;j++){
                if(nums[i]==nums[j])
                    return true;
            }
        }
        return false;
    }

解法二：哈希表法

  • 第一步：遍历数组
  • 第二步：若哈希表中无该元素，则将遍历的该元素存入哈希表里（利用哈希表的元素唯一的特性）
  • 第三步：若有该元素，则说明有重复元素，返回true
    public boolean containsDuplicate(int[] nums) {
        HashSet<Integer> has= new HashSet<Integer>();
        for(int i=0;i<nums.length;i++){
            if(!has.add(nums[i])){
                return true;
            }
        }
        return false;
    }

解法三：排序法
  • 第一步：对数字从小到大排序
  • 第二步：扫描已排序的数组，每次判断相邻的两个元素是否相等，如果相等则说明存在重复的元素

    public boolean containsDuplicate(int[] nums) {
        Arrays.sort(nums);
        int n = nums.length;
        for (int i = 0; i < n - 1; i++) {
            if (nums[i] == nums[i + 1]) {
                return true;
            }
        }
        return false;
    }
