王道数据结构 - wolai 笔记
👏以王道一道算法题为例，好好利用代码块可以让自己更好的理解代码段。

线性表P17
  1. 从顺序表中删除具有最小值的元素（假设唯一）并由函数返回被删除的元素的值。空出的位置由最后一个元素填补，若顺序表为空，则显示出错信息并退出运行。
    本题为第一次练习，直接阅读答案
    bool Del_Min(sqList &L,ElemType &value)//删除顺序表L中的最小元素节点，并通过引用型参数value返回其值
//若删除成功，则返回true；否则返回false
{
if(L.length==0)
  return false;
 value=L.data[0];
 int pos=0;
 for(int i=1;i<L.length;i++)
   if(L.data[i]<value）
   {
     value=L.data[i];
     pos=i;//最后一个元素填补时，定位依据
    }
 L.data[pos]=L.data[L.length-1];//最后一个元素的下标为L.length-1
 L.length--;
 return true;
 }
